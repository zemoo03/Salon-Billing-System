# Salon Management System

A full-stack, modern web application designed for EVES Spa & Salon to manage their services, personnel, and billing effectively. 

## Features
- **Role-based Dashboards:** Dedicated insights for Admins, Owners, and Staff.
- **Service & Staff Management:** Control your service catalogue and commission rates.
- **Combined Billing:** Check out a customer for multiple services provided by different employees.
- **Staff Analytics:** Transparent tracking of individual employee revenue and salary/incentive payouts.
- **Reporting:** Overall business statistics and service demand reports.
- **Offline Persistence:** Uses zero-latency `localStorage` mock-backend data.

## Technologies
- React 18, React Router v6
- Vite
- TypeScript
- Recharts (Analytics Data)
- Custom CSS System
